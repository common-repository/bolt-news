== Installation ==

Bolt News can be installed like any other wordpress plugin.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/bolt-news` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Bolt News screen to configure the plugin
